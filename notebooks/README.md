# Notebook Notes

This folder contains notes and references for the notebook-based parts of the image classification project.

## Purpose

The notebooks are used for model training, testing, evaluation, and experiment tracking.

## Main Notebook Tasks

- Loading and preparing image data
- Setting up the model architecture
- Training the model
- Checking validation results
- Reviewing incorrect predictions
- Recording experiment outputs

## Notes

Notebook files may be added or updated as the project develops.
